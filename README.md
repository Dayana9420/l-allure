# L'Allure - Hub de Catálogos

Tu plataforma centralizada de catálogos: Vianney, Leonisa, Belcorp y Tu Calzado.

## 🚀 Cómo empezar

1. Instala dependencias: `npm install`
2. Ejecuta desarrollo: `npm run dev`
3. Abre http://localhost:3000
